Internet - TCP/IP模型 - 协议族



100Mbps 每秒传输100M的比特
100MBps 每秒传输100M的字节
1字节 = 8比特
所以100M宽带最高下载速度只能到达12.5M/s


握手机制 ：TCP中要用到三次握手
A请求B
B确认A
A确认请求B
开始发数据


UDP不需要握手，数据传输效率高，一般用于通信，传输网络视频、音频等等。

分层结构，下层为上层提供服务
应用层 - HTTP / SMTP / POP3 / FTP / SSH / QQ
传输层 - TCP/UDP - 端到端传输（数据不传丢，不传错；数据传输速度一定；）
IP层（网络层） - 寻址和路由（一站站的往下传输）；
网络接口层 - 接入网络（分帧（分片进行传输）、校验（数据传错了重新传输））；

HTTP - Hyper-Text Transfer Protocol  超文本传输协议
HTML - Hyper-Text Markup Language    超文本标记语言
常用请求：
- get - 从服务器获取资源(网页、图片、音视频)
- post - 提交数据给服务器
不常用请求：
- delete - 删除服务器上的某个资源
- put - 修改服务器上的某个资源
- patch - 更新服务器上的某个资源
